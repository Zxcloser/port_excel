В терминале заупустить код:
python ./main.py 'from.xls' 'out.xlsx'

где 'from.xls' - путь к ecxel файлу из которого берутся данные,
'out.xlsx' - путь к ecxel файлу формата xlxs в который записываются измененные данные

Можно указывать без параметров, или только с одним.
Пр.:
python ./main.py 'from.xls'
python ./main.py '' 'from.xls'
